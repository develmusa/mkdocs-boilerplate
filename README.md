# MkDocs Boilerplate

A documentation site built with [MkDocs](https://www.mkdocs.org/) and the [Material theme](https://squidfunk.github.io/mkdocs-material/).

## Quick Start

```bash
# Install dependencies
uv sync

# Start development server
mkdocs serve

# Build site
mkdocs build
```

## Documentation

Full documentation is available at [http://localhost:8000](http://localhost:8000) when running the dev server.

See the `docs/` directory for content.
